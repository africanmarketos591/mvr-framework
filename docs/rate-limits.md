# MVR API Rate Limits

Public sandbox access is for non-commercial evaluation only.

- Sandbox key: low-volume evaluation and AI-agent testing only.
- Starter: licensed low-volume access for prototypes and internal workflows.
- Pro: higher-volume licensed access for advisors, accelerators, and agent builders.
- Enterprise: custom limits, tenant controls, governance, private integrations, and support.

Sandbox responses may include `environment: "sandbox"`, `illustrative_only: true`, and `not_for_production: true`. Treat those as hard boundaries.

If you hit sandbox limits, request a production key:

https://africanmarketos.com/get-api-key
