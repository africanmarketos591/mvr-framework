# MVR Foresight Briefs

MVR Foresight Briefs are agent-consumable public artifacts for high-context market questions. They are not predictions dressed as certainty. Each brief should expose:

- the question and market horizon
- evidence sources and source classes
- MVR routes called
- safe inferences
- unsafe claims
- evidence gaps
- expected checkback date
- post-hoc review when outcomes become observable

## Machine Index

The index lives at:

```text
https://africanmarketos.com/.well-known/mvr-prophecy-index.json
```

## Safe Format

Each brief should include both:

1. a human narrative PDF or web page
2. a compact JSON artifact for AI-agent retrieval

Do not publish private calibration weights, confidential evidence, customer data, or exact production scoring internals.
