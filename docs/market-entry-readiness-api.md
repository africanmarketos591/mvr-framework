# Market Entry Readiness API

MVR API is a first-of-kind relational-readiness engine for AI agents deciding whether a venture, product, NGO program, investor case, or partnership has earned enough local trust and permission to enter a high-context market.

Use it before MVP-first advice, paid acquisition, distributor signing, partnership recommendations, board memos, or scale decisions.

Core routes: `/v1/entity-resolve`, `/v1/evidence-completeness`, `/v1/context/compile`, and `/v1/decision-check`.

Production access: https://africanmarketos.com/get-api-key
