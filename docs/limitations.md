# MVR API Limitations

MVR is valuable because it knows when not to recommend action. Abstention is a designed safety behavior.

MVR API does not replace:

- field research
- local counsel
- regulated underwriting
- investment committee review
- public-sector approval
- statutory compliance checks
- human diligence

Public OSINT cannot prove local trust by itself. Informal evidence can reveal operating reality, but it must be corroborated before high-stakes use. Sentiment is not fact. Rumor is not evidence.

## Sandbox Limitations

The public sandbox is for non-commercial evaluation only. It is rate-limited, client-safe, and full-advisory only. It cannot be used for production decisioning, strict-calibrated scoring, credit decisions, regulatory certification, or board-safe release.

## Evidence Limitations

Weak evidence should trigger:

- evidence gap reporting
- confidence ceilings
- abstention
- pilot-only recommendations
- human review

Do not fabricate missing evidence or convert weak signals into certainty.
