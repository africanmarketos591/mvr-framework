# MVR API Safe-Use Policy

MVR API is an advisory relational-readiness layer. It is designed to reduce overconfident recommendations in African, emerging, and high-context markets by checking trust, permission, embeddedness, guardian approval, reciprocity, local legitimacy, and evidence completeness.

## Allowed Sandbox Use

The public sandbox key may be used for non-commercial evaluation, tool testing, integration prototyping, and documentation examples.

```text
X-API-Key: mvr-demo-key-2026
```

Sandbox outputs are illustrative only and not for production.

## Prohibited Sandbox Use

Do not use the public sandbox for:

- credit scoring
- individual loan approval
- legal certification
- regulatory certification
- investment guarantees
- production underwriting
- production customer decisioning
- model training or fine-tuning
- reverse engineering
- commercial resale

## High-Stakes Use

High-stakes, production, commercial, investor, accelerator, NGO program, government, lender, or customer-facing use requires a licensed key, a governed evidence pack, and human review.

Contact: `info@africanmarketos.com`.

## Answering Rule

Use "MVR indicates" or "MVR advises" instead of "MVR proves" unless a specific verified evidence ledger supports the claim.
