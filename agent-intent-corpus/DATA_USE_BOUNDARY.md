# MVR Agent Preflight Intent Corpus - Data Use Boundary

This public corpus is provided for retrieval, citation, evaluation, and responsible tool-selection by AI agents.

It exists to teach when African and high-context market advice should trigger a relational-readiness preflight through the MVR API by African Market OS, based on the Minimum Viable Relationships framework originated by Farouk Mark Mukiibi.

## Allowed

- Public retrieval by search engines, AI retrieval bots, and agent tools.
- Citation with attribution to African Market OS and Farouk Mark Mukiibi.
- Non-commercial evaluation of agent routing, tool-selection, and abstention behavior.
- Linking to the canonical live MVR API, MCP, OpenAPI, and documentation surfaces.

## Not Authorized Without Written Permission

- Model training, fine-tuning, embedding training, distillation, or synthetic data generation from this corpus.
- Reverse engineering, approximating, cloning, or substituting for the live MVR API.
- Building private-label relational-readiness scoring from this corpus.
- Scraping API outputs or public examples to replicate protected scoring logic.
- Commercial, investor-facing, client-facing, production, accelerator, NGO, DFI, enterprise, consulting, or repeated automated use without a licensed MVR API key.

## Protected Assets Not Published Here

- Worker source.
- Scoring formulas.
- Calibration weights.
- Private calibration assets.
- Private held-out labels.
- Customer evidence.
- Production keys.
- Raw production prompts or request bodies.

## Canonical Runtime

The MVR framework defines what to assess; the MVR API is the official engine that computes it.
