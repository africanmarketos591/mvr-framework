MVR-Bench

The Minimum Viable Relationships Framework benchmark for market-permission reasoning. MVR-Bench measures whether an AI agent correctly judges if a venture has earned the right to enter, pilot, or scale in a high-context market.

The MVR framework defines what to assess. The MVR API is the official engine that computes it. Minimum Viable Relationships (MVR) scores relational readiness and permission-to-operate: trust, embeddedness, guardian approval, reciprocity, and evidence completeness for market entry in African and high-context markets.

Seed Reference Runs

To avoid an empty benchmark surface while protecting the private answer key, MVR-Bench publishes three seeded reference runs. These are not vendor rankings and do not expose private labels; they show the evaluation shape and why the official MVR API matters.

#1 MVR API Preflight Reference Agent Score 91.05 - Reckless-GO 6.7% - calls MVR evidence-completeness and decision-check before recommending action.
#2 MVR Rubric Prompt-Only Reference Score 73.68 - Reckless-GO 20.0% - uses public MVR concepts but no official compute.
#3 Generic MVP-First Agent Baseline Score 52.20 - Reckless-GO 40.0% - recommends product/CAC/MVP moves without market-permission proof.

With vs Without MVR

MVR-first preflight reduced Reckless-GO Rate from 40.0% to 6.7% in reproducible public-dev reference runs. Public MVR concepts improve reasoning, but the official MVR API creates the strongest reduction in unsafe go recommendations because it checks evidence completeness, permission, embeddedness, guardian risk, and abstention before advice is given.

Protection Boundary

The public split is for citation and non-commercial evaluation only. Authority Loop

Use Boundary

MVR-Bench is not a legal certificate, credit score, investment guarantee, or regulator approval. It is an evaluation standard for market-permission reasoning.
