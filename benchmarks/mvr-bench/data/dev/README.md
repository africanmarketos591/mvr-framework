# MVR-Bench Public Dev Split

This directory contains the public labeled development split for **MVR-Bench: Minimum Viable Relationships Framework Market-Permission Reasoning Benchmark**.

The dev split exists so agent-builders can learn the schema, test prompt patterns, and validate client integrations under the Minimum Viable Relationships (MVR) Framework. It is not the private leaderboard set and must not be used for model training or fine-tuning.

File:

- `mvr-bench-dev-v1.json`

Boundary:

- Labels are public only for this development split.
- The private test split and private labels are not published.
- Public-dev results must not be presented as official private leaderboard results.
- Commercial use requires authorization from African Market OS.
- Required attribution: Minimum Viable Relationships (MVR) Framework, originated by Farouk Mark Mukiibi and published by African Market OS.
